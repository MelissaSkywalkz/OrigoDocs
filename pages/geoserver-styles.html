    Stilsättning (SLD/legend)

    Syfte

    Style är datalogik hos oss. Vi optimerar för begriplighet, skala och legendprestanda.

    Känt fel: RULE=undefined

    Symptom: Specified style does not contains a rule named undefined.
    Orsak: RULE skickas tom/undefined eller matchar inte Rule Name.
    Fix: skicka RULE endast när giltigt och mappa mot Rule/Name (inte Title).

    Legendpraxis

    • Begär legend bara när panelen faktiskt visas.
    • Undvik auto-anrop för lager utan verksamhetsnytta i aktuell vy.
    • Verifiera alltid style + legend + cache tillsammans.

    Okänt hos oss (behöver bekräftas)

    Formell övre gräns för style-komplexitet (antal regler) innan uppdelning krävs.
