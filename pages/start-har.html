Onboarding

Ny utvecklare: start här

Syfte

Snabb väg in i hur vi faktiskt jobbar med Origo/GeoServer/GWC på srv-origo01.

Dag 1: få upp lokal miljö

  1. Kloning vid ny dator (internt repo): git clone //srv-origo01/Git/Origo.git Origo.
  2. Alternativt mot GitHub-org: git clone https://github.com/SkovdeGIS/origo.git.
  3. Installera beroenden: npm install.
  4. Starta devserver: npm start (standard http://localhost:9966).

Första veckan: rätt läsordning

  1. Origo i vår miljö (källor, grupper, projektion).
  2. GeoServer: attribut & filtrering.
  3. GeoServer: stilsättning.
  4. GWC: caching.
  5. Felsökningshandbok.

Kritiska URL:er att känna till

  • GeoServer admin: https://srv-origo01.kommun.skovde.se/geoserver/web
  • WMS/WFS endpoint: https://srv-origo01.kommun.skovde.se/geoserver/ows/
  • CSW: https://srv-origo01.kommun.skovde.se/geoserver/csw/
  • Intern Origo: https://srv-origo01.kommun.skovde.se/
  • Publik Origo: https://origo.skovde.se
