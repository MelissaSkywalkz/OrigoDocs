WMS GetMap test (liten bbox, låg last)

curl -G 'https://srv-origo01.kommun.skovde.se/geoserver/intern/wms' \
  --data-urlencode 'service=WMS' \
  --data-urlencode 'version=1.1.1' \
  --data-urlencode 'request=GetMap' \
  --data-urlencode 'layers=intern:fb_byggnad_y' \
  --data-urlencode 'styles=' \
  --data-urlencode 'format=image/png' \
  --data-urlencode 'transparent=true' \
  --data-urlencode 'srs=EPSG:3008' \
  --data-urlencode 'bbox=172500,6477600,173500,6478600' \
  --data-urlencode 'width=512' \
  --data-urlencode 'height=512'

GWC seed/troubleshoot-checklista

  1. GetMap fungerar?
  2. Legend fungerar?
  3. Rätt layer/truncate?
  4. Rätt nivåspann i seed?
  5. Verifierat i både WMTS-request och Origo?
