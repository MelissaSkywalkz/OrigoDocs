Quick recipes (copy/paste)

Syfte

Snabba testanrop för verifiering med låg last, innan större felsökning eller seedning.

WMS GetMap (liten bbox)

curl -G 'https://<host>/geoserver/ows' \
  --data-urlencode 'service=WMS' \
  --data-urlencode 'version=1.1.1' \
  --data-urlencode 'request=GetMap' \
  --data-urlencode 'layers=intern:exempel_lager' \
  --data-urlencode 'styles=' \
  --data-urlencode 'format=image/png' \
  --data-urlencode 'transparent=true' \
  --data-urlencode 'srs=EPSG:3006' \
  --data-urlencode 'bbox=674032,6580821,674532,6581321' \
  --data-urlencode 'width=512' \
  --data-urlencode 'height=512'

WMTS URL-exempel

https://<host>/geoserver/gwc/service/wmts?SERVICE=WMTS&REQUEST=GetTile&VERSION=1.0.0
&LAYER=intern:exempel_lager&STYLE=&TILEMATRIXSET=EPSG:3006
&TILEMATRIX=EPSG:3006:10&TILEROW=536&TILECOL=292&FORMAT=image/png

GetLegendGraphic + Origo

https://<host>/geoserver/ows?service=WMS&request=GetLegendGraphic
&layer=intern:exempel_lager&format=image/svg+xml&transparent=true

I Origo-lager där vi vill minimera last: sätt legend avstängd som default och aktivera endast för prioriterade lager.

GWC seed/troubleshoot-checklista

  1. GetMap OK med liten bbox?
  2. GetLegendGraphic OK?
  3. Truncate utförd för rätt lager/grupp?
  4. Seedning gjord för rätt zoomnivåer?
  5. Verifikation i både WMTS-anrop och Origo-klient?
