WMS GetMap (låg last, liten bbox)

curl -G 'https://srv-origo01.kommun.skovde.se/geoserver/ows/' \
  --data-urlencode 'service=WMS' \
  --data-urlencode 'version=1.1.1' \
  --data-urlencode 'request=GetMap' \
  --data-urlencode 'layers=publik:fb_kommunal_fastighet_y' \
  --data-urlencode 'format=image/png' \
  --data-urlencode 'transparent=true' \
  --data-urlencode 'srs=EPSG:3008' \
  --data-urlencode 'bbox=172500,6477600,173500,6478600' \
  --data-urlencode 'width=512' \
  --data-urlencode 'height=512'

GWC seed-checklista

  1. Välj EPSG:3008 gridset.
  2. Sätt bbox (inte tom).
  3. Kör truncate före seed.
  4. Verifiera i klient att WMTS används.
