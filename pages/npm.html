    npm

    npm & plugins

    Verktyg

    Installera plugins via npm

    npm install @leverantor/origo-plugin
npm install @leverantor/origo-plugin@1.2.3
    • Använd exakta versioner för stabilitet.
    • Notera eventuella peer dependencies.

    Bygg in i ett statiskt projekt

    Om pluginen kräver byggsteg: använd en enkel bundler (t.ex. Vite) och generera statiska filer. Kontrollera i pluginets README om bundling krävs. 

    # exempel på enkel setup
npm create vite@latest origo-demo
cd origo-demo
npm install
npm run build

    Snabba kommandon

    Init & install

    npm init -y
npm install paket
npm install paket --save-dev

    Uppdatera & audit

    npm update
npm audit
npm audit fix

    Versioner & semver (kort)

    • ^1.2.3 = tillåter mindre uppdateringar.
    • ~1.2.3 = tillåter patchar.
    • 1.2.3 = låst exakt version.

    Lockfile

    Commit alltid package-lock.json för reproducibla byggen.
