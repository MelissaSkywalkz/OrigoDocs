Vad är Layermanager?

Layermanager är ett Origo-plugin som hjälper användare att hitta och lägga till lager från en katalogtjänst (t.ex. GeoServer eller CSW). TODO: verifiera exakt stöd i repo.

Så här lägger du till Layermanager i Origo

  1. Installera pluginet (npm eller inkluderad bundle).
  2. Aktivera pluginet i Origo-konfigurationen.
  3. Peka pluginet mot katalog/GeoServer-URL.

Exempel: Layermanager i konfig (pseudo-JSON)

{
  "plugins": [
    {
      "name": "layermanager",
      "options": {
        "url": "https://geoserver.example.com/geoserver",
        "types": ["tema-1", "tema-2"],
        "onlyAddableLayersBtn": true
      }
    }
  ]
}

Vad betyder options?

  • url: adress till katalog/WMS/CSW som Layermanager läser från.
  • types: filter för tematyper (t.ex. kategorier).
  • onlyAddableLayersBtn: visar endast lager som går att lägga till.

Checklistor

  1. Testa URL:en i webbläsare (får du svar?).
  2. Verifiera att Layermanager-pluginet laddas (ingen 404).
  3. Kontrollera att lagren visas i UI.

Vanliga fel

  • Tom lista → fel URL eller saknade behörigheter.
  • 404 på plugin-script → fel build/filnamn.
  • Inga filter-träffar → fel types-värden.
