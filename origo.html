Hur Origo pratar med GeoServer

Origo är klienten som visar kartan. GeoServer är tjänsten som levererar data. Mellan dem används oftast WMS (bilder) och WFS (features).

  • Origo hämtar data via en WMS/WFS-endpoint.
  • GeoServer svarar med kartbilder eller objekt.
  • TODO: verifiera hur endpoints anges i Origo-konfiguration.

Origo i praktiken (kort)

  • WMS: bra för snabb visning och bakgrundslager.
  • WFS: behövs för klick, filter och clustering.
  • Styling: ändras i GeoServer, syns i Origo via WMS/tiles.

Begreppslista

Layer
Visuellt lager i kartan (WMS, WMTS, XYZ, vektor etc.).
Source
Datakälla som layern hämtar data från.
Style
Hur ett lager ska se ut (färger, symboler).
Control
UI-komponent, t.ex. zoom eller skalstock.
Plugin
Extern modul som bygger ut funktionalitet.
