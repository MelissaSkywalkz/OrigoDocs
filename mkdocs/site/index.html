MkDocs preview-path fungerar

Den här sidan ligger i /mkdocs/site/ för att undvika 404 på GitHub Pages root-spåret.

Den fulla Material-previewn byggs av workflow: .github/workflows/mkdocs-preview.yml.

  • Repo root: /OrigoDocs/
  • MkDocs källfiler: /OrigoDocs/mkdocs/docs/
  • Preview-start i markdown: /OrigoDocs/mkdocs/docs/index.md

För riktig MkDocs-rendering: kör Actions-jobbet Deploy MkDocs Preview eller lokalt mkdocs build -f mkdocs.yml --strict.
